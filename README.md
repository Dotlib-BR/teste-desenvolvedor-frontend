# Resolução teste desenvolvedor frontend

Seguem instruções para rodar o projeto


```sh
# instala as dependências npm
npm install    

# inicia o json-server
npm run server

# inicializa a página 
npm start
```

## Pacotes utilizados

- Axios (para requisições de API)
- Json server (para carregar a API)
